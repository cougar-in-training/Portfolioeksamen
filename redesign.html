Link til redesign af Rust&Kaffen

I denne opgave skulle vi redesigne den valgte virksomhed's hjemmeside, som vi også lavede virksomhedspræsentations video til.
Vi startede med at analysere den eksisterende hjemmeside for evt. fejl og mangler. Vores gruppe blev hurtigt enige om at hjemmesiden ikke fangede cafens stemning eller ejerens personlighed.
Tankerne bag vores redesign skulle reflekterer dagligstue-stemningen som Rust&Kaffen besider vha. varme jordnære farver.

Vores gruppe blev desværre hårdt ramt af sygdom fra 2.dag. Så alt HTML og CSS blev lavet af mig. Jeg hjalp det ene gruppemedlem med at skrive rapporten weekenden op til afleveringen.

    Værktøjer brugt:
  • Photoshop
  • InDesign
  • HTML
  • CSS
